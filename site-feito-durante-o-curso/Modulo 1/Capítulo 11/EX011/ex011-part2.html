Reproduzindo Áudio

Vamos aprender a reproduzir áudios em HTML5.

Na Aula 3 foi apresentado como inserir audio na página, o código é esse: 

        
        <audio preload="metadata" autoplay controls>
            <source src="midia/guanacast-33.mp3" type="audio/mpeg">
            <source src="midia/guanacast-33.ogg" type="audio/ogg">
            <source src="midia/guanacast-33.wav" type="audio/wav"> <--Tentar evitar arquivos Wav, pois, são arquivos muito pesados. Arquivos com qualidades boas são para Download e não Navegação!-->
            <Infelizmente seu navegador não consegue reproduzir áudio. <a href="midia/guanacast-33.mp3">Clique Aqui para Baixar o Arquivo><>
        </audio>
Este código abrimos a tag audio onde nela temos os seguintes atributos, preload, autoplay e controls. Preload, é um atributo importantissimo, pois ele indica ao navegador como deve carregar o áudio, nele, podemos ter valores como: 
        
none: Não carrega o áudio até que o usuario inicie a reprodução
metadata: Carrega apenas os metadados (como duração e faixa de áudio), sem baixar o conteúdo.
auto: Tenta carregar o áudio automaticamente, dependendo da situação (conexão, dispositivos, etc.).
Temos também o autoplay, onde o áudio ira começar tocar sozinho, quando pronto. Controls exibe os controles padrões do player de áudio, iniciar e pausar, aumentar e diminuir o volume. Dentro da tag audio, temos outra tag, a source, onde é utilizado para especificar múltiplas fontes de mídia, nela temos os atributos, src e type. Src irá definir o caminho da mídia e type indica o tipo MIME do arquivo, que ajuda o navegador a identificar o formato. Passando para os formatos, os mais comuns aceitos pelos navegadores são, MP3, WAV, OGG, AAC. No código a estrutura de sources importa, pois irá carregar um pós o outro, exemplo, se o mpeg não carregar, ele irá tentar o ogg assim por diante, no caso foi definido por tamanho do mais leve ao mais pesado.
